## Apply Filters to SQL Queries

Demonstrated how SQL filtering techniques can be used to investigate security-related data. Used AND, OR, NOT, and LIKE operators to analyze login attempts and employee records for security monitoring and system updates.

**Skills demonstrated:**
- Filtering by time and date
- Using AND / OR conditions
- Using NOT to exclude results
- Pattern matching with LIKE (%)
