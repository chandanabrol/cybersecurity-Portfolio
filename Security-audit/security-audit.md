## Security Audit – Botium Toys

**Objective:**  
Conducted a simulated security audit to evaluate security controls, identify risks, and review compliance gaps.

**Key Areas Reviewed:**
- Asset management
- Access controls
- Data protection
- Monitoring and incident readiness
- Compliance with PCI DSS, GDPR, and SOC standards

**Outcome:**
Identified multiple control gaps, including lack of least privilege, encryption, backups, and intrusion detection. Provided practical recommendations to improve the organization’s security posture.
