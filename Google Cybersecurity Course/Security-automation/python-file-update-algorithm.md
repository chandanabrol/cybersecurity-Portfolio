# Python File Update Algorithm

This project demonstrates a Python algorithm used to update an allow list of IP addresses that control access to restricted systems.

## Scenario

A healthcare organization stores approved IP addresses in an allow_list.txt file. A remove_list identifies IP addresses that should no longer have access.

The Python script checks whether any IP addresses in the remove list exist in the allow list and removes them automatically.

## Python Concepts Used

- File handling with `open()` and `with`
- `.read()` and `.write()` methods
- `.split()` and `.join()` methods
- `for` loops
- `.remove()` list operation

## Skills Demonstrated

- Python automation
- Access control management
- File parsing
- Security scripting
