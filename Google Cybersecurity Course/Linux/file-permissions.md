## Use Linux Commands to Manage File Permissions

Demonstrated how Linux file permissions are examined and modified to ensure proper authorization. Used ls -la to inspect permissions and chmod to remove unauthorized access for files, hidden files, and directories.

**Skills demonstrated:**
- Interpreting Linux permission strings
- Managing file and directory permissions
- Securing hidden files
- Applying least privilege principles
